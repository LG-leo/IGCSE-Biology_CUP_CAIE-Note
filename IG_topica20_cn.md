### 📌 一、Paper 2 – 高频考点与答题关键词 (选择题)

| 考点类别 | 真题关键词 (Question Keyword) | 答题关键词 (Answer Key) |
| :--- | :--- | :--- |
| **生物多样性定义** | 以下哪个是对生物多样性 (biodiversity) 的准确定义？ | The number of different species that live in an area. |
| **栖息地破坏原因** | 以下哪项是栖息地破坏的原因？ | 城市化 / 农业扩张 / 开采自然资源 / 污染。 |
| **单一栽培负面影响** | 大规模单一栽培 (monoculture) 的缺点是什么？ | 降低遗传多样性 / 增加病虫害风险 / 耗竭土壤养分 / 降低生物多样性。 |
| **森林砍伐的影响** | 以下哪项最不可能 (least likely) 是森林砍伐 (deforestation) 的结果？ | **生物多样性增加** (An increase in biodiversity)。 |
| **森林砍伐的直接原因** | 森林砍伐的主要原因是什么？ | 农业用地扩张 / 城市化 / 伐木。 |
| **不可生物降解塑料的影响** | 塑料污染对水生和陆地生态系统的影响包括？ | 动物摄入或缠绕 / 分解为微塑料 / 释放毒素。 |
| **污染物识别** | 以下哪种污染物会导致水体富营养化 (eutrophication)？ | 未经处理的污水 / 化肥 (富含硝酸盐和磷酸盐)。 |
| **可持续资源的定义** | 什么是可持续资源 (sustainable resource)？ | A resource which is produced as rapidly as it is removed from the environment so that it does not run out。 |
| **保护渔业的方法** | 以下哪项有助于维持鱼类资源 (fish stocks)？ | 建立保护区 / 设置禁渔期 / 控制网眼大小 / 配额管理。 |
| **保护濒危物种的方法** | 以下哪项是保护濒危物种 (endangered species) 的方法？ | 人工繁育计划 / 种子库 / 教育和栖息地保护。 |
| **温室效应增强** | 人类活动导致大气中哪种气体浓度增加，导致温室效应增强？ | 二氧化碳 (Carbon dioxide, CO₂) 和甲烷 (Methane, CH₄)。 |
| **化石燃料燃烧的影响** | 燃烧化石燃料 (combustion of fossil fuels) 的主要环境后果是什么？ | 增加大气中的二氧化碳，增强温室效应。 |
| **过度捕捞的定义** | 以下哪项是对过度捕捞 (overfishing) 的最准确定义？ | 以超过种群自然补充的速度捕捞鱼类。 |
| **外来物种的影响** | 引入外来物种 (introduced species) 对本地生态系统的主要负面影响是什么？ | 与本地物种竞争资源，可能没有天敌，导致本地物种减少或灭绝。 |

### 📝 二、Paper 4 – 典型题目与答题模板 (理论题)

| 题型 | 真题关键词 | 答题模板 (按评分标准) |
| :--- | :--- | :--- |
| **森林砍伐的影响** | Explain the undesirable effects of deforestation. | **Reduced biodiversity**: loss of habitats for many species, leading to extinction. **Loss of soil**: tree roots hold soil together; without them, soil is easily eroded by rain and wind. **Flooding**: less water absorbed by trees and soil, increased surface runoff leads to flooding. **Increase in atmospheric CO₂**: fewer trees for photosynthesis and burning/rotting trees release CO₂. |
| **富营养化过程 (6分题)** | Explain the process of eutrophication. | 1. Excess nitrates/phosphates from fertiliser/sewage enter water. 2. Algae and aquatic plants grow rapidly (algal bloom). 3. Algae block sunlight → submerged plants die. 4. Dead matter decomposed by bacteria (decomposers). 5. Bacteria increase → aerobic respiration uses up oxygen. 6. Dissolved oxygen decreases → fish and other organisms die. |
| **温室效应增强** | Explain the enhanced greenhouse effect and how it leads to climate change. | Human activities (burning fossil fuels, deforestation, agriculture) release greenhouse gases (CO₂, methane). These gases trap infrared (thermal) radiation in the atmosphere. This causes the enhanced greenhouse effect, leading to global warming and climate change (rising sea levels, extreme weather). |
| **可持续森林管理** | Describe how forests can be conserved sustainably. | **Education**: teaching the importance of forests and sustainable use. **Protected areas**: establishing national parks and reserves. **Quotas**: limiting the amount of timber harvested. **Replanting**: planting new trees to replace those cut down. |
| **可持续渔业管理** | Describe how fish stocks can be conserved sustainably. | **Education**: educating fishermen about sustainable practices. **Closed seasons**: banning fishing during breeding seasons. **Protected areas**: creating marine protected areas. **Controlled net types and mesh size**: using nets that allow small/young fish to escape. **Quotas**: setting limits on total catch. **Monitoring**: regularly assessing fish populations. |
| **过度捕捞的影响** | Explain how overfishing affects the ecosystem. | Removing a large number of fish reduces food availability for predators (e.g., seabirds, larger fish). It may cause population increase in prey species (e.g., zooplankton). This disrupts the balance of the food web and can lead to collapse of fish stocks. |
| **保护濒危物种的方法** | Describe how endangered species can be conserved. | **Monitoring and protecting species and habitats**; **Education** to raise awareness; **Captive breeding programmes** to increase population numbers; **Seed banks** to preserve genetic diversity of plant species. |
| **单一栽培的优缺点** | Discuss the advantages and disadvantages of large-scale monocultures of crop plants. | **Advantages**: efficient for planting, management, and harvesting using machinery; high yields. **Disadvantages**: lack of genetic variation makes crops vulnerable to disease or pests; reduces biodiversity; depletes soil nutrients. |
| **生物富集** | Explain how pesticides can accumulate in food chains. | Pesticides are non-biodegradable and persist in the environment. They are absorbed by plants or small organisms. When animals eat these organisms, the pesticide accumulates in their bodies. Concentration increases at each trophic level (biomagnification), causing harm to top predators. |
| **减少污染的方法** | Suggest ways to reduce pollution from fertilisers. | Use organic fertilisers instead of chemical fertilisers; apply fertilisers in appropriate amounts (not excess); use buffer strips of vegetation to absorb runoff; plant cover crops to reduce nutrient leaching. |

### ⚠️ 三、Paper 2 常见陷阱（基于真题失分点）

| 陷阱 | 正确理解 | 错误选项特征 |
| :--- | :--- | :--- |
| **混淆温室效应与增强温室效应** | 温室效应是自然过程，使地球保持温暖；**增强温室效应**由人类活动（燃烧化石燃料、森林砍伐）导致，引起全球变暖。 | 选项中写“greenhouse effect is caused by humans” (混淆两者)。 |
| **混淆富营养化的原因和结果** | 富营养化由**过量营养盐（硝酸盐、磷酸盐）** 引起，不是由氧气减少引起。氧气减少是**结果**，不是原因。 | 选项中写“lack of oxygen causes eutrophication”。 |
| **混淆化肥和杀虫剂的环境影响** | 化肥 → 富营养化；杀虫剂 → 生物富集/生物放大。 | 选项中写“fertilisers cause bioaccumulation”。 |
| **混淆森林砍伐对CO₂的影响** | 森林砍伐**增加**大气CO₂（光合作用减少 + 燃烧/腐烂释放CO₂）。 | 选项中写“deforestation decreases CO₂”。 |
| **混淆可持续资源与不可再生资源** | 可持续资源（如森林、鱼类）可在消耗的同时**再生**；化石燃料是**不可再生**资源。 | 选项中写“fossil fuels are sustainable resources”。 |
| **混淆人工授精和体外受精** | 人工授精 (AI)：精子注入雌性体内，**体内受精**；体外受精 (IVF)：精卵在体外融合，**体外受精**，再将胚胎植入子宫。 | 选项中混淆两者定义。 |
| **混淆杀虫剂和除草剂** | **杀虫剂** (insecticides) 杀死昆虫；**除草剂** (herbicides) 杀死杂草/植物。 | 选项中写“insecticides kill weeds”。 |
| **混淆灭绝的原因** | 栖息地破坏、过度捕捞、污染、气候变化、引入外来物种都可能导致灭绝。 | 选项中只列出一个原因，或列出无关原因。 |
| **错误认为所有污染物都不可生物降解** | 一些污染物（如污水中的有机物）可被微生物**分解**；塑料和重金属**不可生物降解**。 | 选项中写“all pollutants are non-biodegradable”。 |

### 📌 五、单点考题示例 (Single Examination Point Questions)

| 单点考题 | 答案 |
| :--- | :--- |
| State the name of the process that causes water bodies to become enriched with nutrients. | Eutrophication. |

| State one method of conserving fish stocks. | Closed seasons / quotas / protected areas / controlled mesh size. |
