## 第 2 部分：详细知识点

### 14.1 协调与反应

#### 14.1.1 哺乳动物的神经系统

神经系统是人体快速的通讯系统。它探测刺激（内、外环境的变化），处理信息，并通过电信号协调作出反应。

**中枢神经系统 (CNS)**：
- 由 **脑** 和 **脊髓** 组成。
- 脑作为控制中心：处理信息、储存记忆、作出决策，并协调复杂的反应。
- 脊髓将脑与身体其余部分连接起来，并协调 **反射动作**（快速、自动的反应）。
- 中枢神经系统受颅骨（保护脑）和脊椎骨（保护脊髓）的保护。

**周围神经系统 (PNS)**：
- 由 **脑和脊髓以外的所有神经** 组成。
- 这些神经将电信号传入和传出中枢神经系统。
- **感觉神经元** 将信号从感受器传至中枢神经系统。
- **运动神经元** 将信号从中枢神经系统传至效应器（肌肉和腺体）。

神经系统的作用是 **协调和调节机体功能**。它确保不同的器官和系统以受控、平衡的方式协同工作。

#### 14.1.2 神经元与电信号

- **电信号** 沿 **神经元**（神经细胞）传导，在身体各处快速传递信息。
- 神经元是特化的细胞，能快速、准确地传导神经冲动。
- 冲动沿单一方向传导：从 **树突 → 细胞体 → 轴突 → 轴突末梢**。

**三种神经元**：

| 神经元类型 | 功能 | 特征 |
| :--- | :--- | :--- |
| **感觉神经元** | 将冲动从 **感受器**（如皮肤、眼）传至 **中枢神经系统** | 长树突，细胞体位于中枢神经系统之外 |
| **中间神经元** | 位于 **中枢神经系统内部**；连接感觉神经元和运动神经元 | 轴突短，树突多 |
| **运动神经元** | 将冲动 **从中枢神经系统传至效应器**（肌肉或腺体） | 长轴突，细胞体位于中枢神经系统内 |

#### 14.1.3 反射弧

**反射动作** 是一种 **自动且迅速** 地将刺激与效应器（肌肉和腺体）的反应整合并协调起来的方式。反射是无需思考、快速且具有保护作用的。

一个 **简单的反射弧** 由以下五个部分组成，顺序必须精确：

1. **感受器** — 探测刺激（例如，皮肤中的温度感受器探测到高温物体的热量）
2. **感觉神经元** — 将冲动从感受器传至中枢神经系统（脊髓）
3. **中间神经元** — 位于脊髓内部；将冲动从感觉神经元传递给运动神经元
4. **运动神经元** — 将冲动从脊髓传至效应器
5. **效应器** — 产生反应的肌肉或腺体（例如，肱二头肌收缩使手缩回）

**反射弧为何重要？** 反射之所以快，是因为神经通路经过脊髓而非一直传到大脑——这在危险情况下可节省宝贵的时间。大脑随后才获知发生了动作。

#### 14.1.4 突触（补充内容）

**突触** 是两个神经元之间的连接点。这是一个神经元与下一个神经元进行通讯的部位。

**突触的结构**：
- 第一个神经元（突触前神经元）的末端含有充满 **神经递质分子**（化学信使）的 **囊泡**。
- 两个神经元之间存在一个狭窄的间隙，称为 **突触间隙**（或突触裂）。
- 第二个神经元（突触后神经元）的细胞膜上含有与神经递质形状互补的 **受体蛋白**。

**突触处发生的事件——完整顺序：**
1. 一个电 **冲动** 到达突触前神经元的末端。
2. 该冲动 **刺激神经递质分子** 从 **囊泡** 释放进入突触间隙（囊泡与细胞膜融合，通过胞吐作用释放其内容物）。
3. 神经递质分子 **扩散通过** 狭窄的突触间隙。
4. 神经递质分子与突触后神经元细胞膜上的 **受体蛋白结合**。
5. 这种结合 **在突触后神经元中激发了一个新的电冲动**。
6. 然后，冲动沿这第二个神经元传导下去。

**突触为何重要——单向传导：**
- 突触确保冲动仅沿 **一个方向** 传导。
- 这是因为只有突触前神经元含有携带神经递质的囊泡，也只有突触后神经元拥有互补的受体蛋白。
- 这种单向传导对神经系统作为一个有组织、有方向的通讯网络发挥功能至关重要。

### 14.2 感觉器官

#### 14.2.1 感觉器官与受体细胞

**感觉器官** 是对 **特定刺激** 作出反应的 **受体细胞** 群。

| 感觉器官 | 探测的刺激 | 受体类型 |
| :--- | :--- | :--- |
| **眼** | 光 | 视网膜中的感光细胞（光感受器） |
| **耳** | 声音和平衡 | 声音感受器 |
| **皮肤** | 触、压、温度 | 触觉和温度感受器 |
| **鼻** | 空气中的化学物质（气味） | 嗅觉感受器 |
| **舌** | 食物中的化学物质（味道） | 味觉感受器 |

每个受体细胞都特化，只检测一种特定类型的刺激。当受体探测到其刺激时，会产生电信号，沿感觉神经元传至中枢神经系统。

#### 14.2.2 人眼——结构与功能

| 结构 | 位置 | 功能 |
| :--- | :--- | :--- |
| **角膜** | 眼睛前方的透明部分 | **折射（弯曲）** 入射光线，使光线进入眼内 |
| **虹膜** | 瞳孔周围的彩色环 | 通过调节瞳孔大小 **控制进入眼内的光量** |
| **瞳孔** | 虹膜中央的黑色圆孔 | 光线进入眼内的通道 |
| **晶状体** | 位于虹膜和瞳孔之后 | 通过改变形状 **将光线聚焦在视网膜上** |
| **视网膜** | 衬于眼球后壁的内层 | 含有感光细胞（**视杆细胞** 和 **视锥细胞**），探测光并产生冲动 |
| **视神经** | 从眼球后部延伸至脑 | 将电信号 **从视网膜传至大脑** |
| **盲点** | 视神经离开视网膜处 | 此处无光感受器细胞；该小范围内无法检测到图像 |

#### 14.2.3 瞳孔反射

**瞳孔反射** 是一种自动反应，通过改变瞳孔直径来控制进入眼内的光量。它保护视网膜免受强光损伤，并帮助眼睛在不同光照条件下获得最佳视觉。

**从拮抗肌作用解释瞳孔反射（补充内容）**：

| 光照条件 | 环行肌 | 辐射肌 | 瞳孔直径 | 进入的光量 |
| :--- | :--- | :--- | :--- | :--- |
| **强光** | **收缩** | 舒张 | **缩小**（变小） | 减少光线进入——保护视网膜 |
| **弱光** | 舒张 | **收缩** | **扩大**（变大） | 增加光线进入——改善视力 |

- 虹膜中的环行肌和辐射肌是 **拮抗肌**——它们以相反方向工作以控制瞳孔大小。
- 当环行肌收缩时，瞳孔直径减小；当辐射肌收缩时，瞳孔被拉开变大。

#### 14.2.4 调节（补充内容）

**调节** 是指眼睛通过改变 **晶状体** 的形状，对不同距离的物体进行聚焦的过程。它由 **睫状肌** 和 **悬韧带** 控制。

**视远物时：**
- **睫状肌舒张**（它们变宽，睫状肌环的直径增加）。
- 这使 **悬韧带被拉紧**（绷直）。
- 拉紧的悬韧带将晶状体拉成 **较薄（较扁平）、凸度较小的形状**。
- 光线被折射（弯曲）得较少，远处物体的图像聚焦在视网膜上。

**视近物时：**
- **睫状肌收缩**（肌肉环的直径变小）。
- 这 **释放了悬韧带上的张力**，悬韧带变得松弛。
- 有弹性的晶状体变成 **更厚、更凸（更圆）的形状**。
- 光线被强烈折射，近处物体的图像聚焦在视网膜上。

#### 14.2.5 视杆细胞与视锥细胞（补充内容）

视网膜含有两种感光细胞：**视杆细胞** 和 **视锥细胞**。

| 特征 | 视杆细胞 | 视锥细胞 |
| :--- | :--- | :--- |
| **功能** | 探测 **光强度**（黑白视觉）；对 **夜视** 具有更高的灵敏度 | 探测 **颜色**；三种类型分别吸收不同颜色的光（红、绿、蓝），用于 **色觉** |
| **灵敏度** | **灵敏度高**，对弱光敏感 | **灵敏度较低**；需明亮光线才能工作 |
| **分布** | 分布于 **整个视网膜**，中央凹缺失 | **集中于中央凹**（视网膜中心区域，此处光线聚焦以实现最清晰的视觉） |

**中央凹（补充内容）**：
- 中央凹是视网膜中心的一个小凹陷。
- 它含有非常高密度的 **视锥细胞**，且 **无视杆细胞**。
- 它是光线聚焦以实现 **最清晰、最细致的色觉** 的区域。

### 14.3 激素

#### 14.3.1 什么是激素？

**激素** 是一种 **化学物质**，由 **内分泌腺** 产生，由 **血液** 运输，能改变 **一个或多个特定靶器官** 的活动。

激素的主要特征：
- 它们是直接分泌到血液中的化学信使。
- 它们随血液循环流遍全身，但只影响具有与之互补的受体蛋白的 **特定靶器官**。
- 它们产生的反应通常比神经冲动 **更慢，但持续时间更长**。
- 它们帮助控制生长、代谢、情绪、生殖和稳态等过程。

#### 14.3.2 内分泌腺及其激素

| 腺体 | 位置 | 分泌的激素 | 主要功能 |
| :--- | :--- | :--- | :--- |
| **肾上腺** | 每个肾脏的上方 | **肾上腺素** | 使身体做好“战斗或逃跑”的准备 |
| **胰腺** | 胃后方，上腹部 | **胰岛素**（以及 **胰高血糖素** — 补充内容） | 降低血糖浓度（胰岛素）；升高血糖（胰高血糖素） |
| **睾丸** | 男性阴囊内 | **睾酮** | 控制男性第二性征的发育（声音低沉、肌肉生长、体毛） |
| **卵巢** | 子宫两侧各一个 | **雌激素** | 控制女性第二性征的发育，并调节月经周期 |

#### 14.3.3 肾上腺素——“战斗或逃跑”激素

**肾上腺素的核心作用**：
- 肾上腺素由肾上腺分泌，在身体可能面临危险或压力的情况下释放。
- 其作用包括：
  1. **加快呼吸频率** —— 为肌肉细胞输送更多氧气用于呼吸作用。
  2. **增加心率** —— 更快地将携带氧气和葡萄糖的血液泵往全身。
  3. **扩大瞳孔直径** —— 让更多光线进入眼内，以获得更清晰的视觉。

**补充内容——肾上腺素额外的代谢作用**：
- **增加血糖浓度** —— 肾上腺素刺激肝脏将糖原分解为葡萄糖并释放入血，为肌肉收缩提供额外燃料。
- 这使得身体为剧烈的体力活动做好准备。

#### 14.3.4 神经调节与激素调节的比较

| 特征 | 神经系统 | 激素（内分泌）系统 |
| :--- | :--- | :--- |
| **信号类型** | 电冲动 | 化学物质（激素） |
| **作用速度** | **极快**（毫秒级） | **较慢**（数秒至数小时） |
| **效应持续时间** | **短暂**（冲动停止即终止） | **较持久**（激素在血液中存留时间更长） |
| **传递途径** | 通过神经元 | 通过血流 |
| **专一性** | 冲动传至特定部位 | 激素随血液流遍全身，但只影响具有特定受体的靶器官 |

### 14.4 稳态

#### 14.4.1 什么是稳态？

**稳态** 是指即使外部环境发生变化，身体仍能 **维持一个恒定的内部环境**。

**为何稳态很重要？**
- 所有细胞都需要一个稳定的环境才能高效工作。
- **酶**——控制代谢的物质——仅在特定条件（适宜的温度、适宜的 pH）下才能发挥最佳作用。
- 没有稳态，细胞受损甚至死亡。

**被稳态控制的关键条件**：

| 受控变量 | 为何重要 | 如何控制 |
| :--- | :--- | :--- |
| **体温** | 酶在约 37°C 时工作最佳 | 体温调节（出汗、战栗、血管舒张/收缩） |
| **血糖水平** | 为呼吸作用提供能量；必须保持稳定 | 由胰岛素和胰高血糖素控制 |
| **含水量** | 细胞需要正确的水平衡以进行渗透作用 | 由肾脏控制（渗透调节） |

#### 14.4.2 负反馈（补充内容）

稳态通过 **负反馈** 系统进行控制：
1. 与正常水平（**设定点**）的偏差被 **感受器** 探测到。
2. 信息被传至 **协调中心**（如脑、脊髓或胰腺）。
3. 协调中心向 **效应器**（肌肉或腺体）发出指令。
4. 效应器产生的反应能 **逆转最初的变化**，使状态重新回到设定点。

**关键原理**：当发生变化时，身体作出 **抵消** 该变化并恢复正常条件的反应。负反馈是所有稳态控制的基础机制。

#### 14.4.3 血糖浓度的控制（补充内容）

血糖浓度必须保持恒定，以确保细胞进行呼吸作用所需的葡萄糖供应稳定。

**当血糖升高时（例如用餐后）：**
1. **胰腺** 检测到血糖升高。
2. 胰腺向血液中释放 **胰岛素**。
3. 胰岛素刺激 **肝脏** 和 **肌肉细胞** 从血液中吸收葡萄糖。
4. 在肝脏中，多余的葡萄糖被转化为 **糖原**（一种不溶性的储存性碳水化合物）并储存起来。
5. 血糖浓度降回正常水平。

**当血糖降低时（例如运动期间、两餐之间）：**
1. **胰腺** 检测到血糖下降。
2. 胰腺向血液中释放 **胰高血糖素**。
3. 胰高血糖素刺激 **肝脏** 将 **糖原** 分解回 **葡萄糖**。
4. 葡萄糖被释放入血。
5. 血糖浓度回升至正常水平。

**胰岛素和胰高血糖素作为一对拮抗激素工作——胰岛素降低血糖，胰高血糖素升高血糖。**

**1 型糖尿病的治疗（补充内容）**：
- 1 型糖尿病是由于胰腺产生极少甚至不产生胰岛素所致。
- 因此，餐后血糖浓度会保持过高。
- 治疗方法包括定期 **注射胰岛素**（通常由患者自行注射），以将血糖降至正常水平。
- 患者还必须小心管理饮食，控制碳水化合物的摄入量。

#### 14.4.4 体温调节（补充内容）

哺乳动物将核心体温维持在大约 **37°C**，这是人体内酶的最适温度。体温调节涉及 **脑**（特别是 **下丘脑**，它相当于身体的恒温器，含有对血液温度敏感的感受器）和 **皮肤**。

**皮肤的结构**（需在图中识别）：毛发、竖毛肌、汗腺、温度感受器、感觉神经元、血管（供皮肤表面毛细血管的小动脉）和脂肪组织。

**当体温升高时（过热）：**

| 反应 | 机制 |
| :--- | :--- |
| **出汗** | 汗腺将汗液分泌到皮肤表面。当汗液 **蒸发** 时，从皮肤带走热量，使身体降温。（蒸发利用身体的热量——汽化潜热。） |
| **血管舒张** | 供皮肤表面毛细血管的小动脉 **舒张**（变宽），使更多血液流经靠近皮肤表面的区域。血液中的热量通过辐射散失到周围环境中。 |
| **毛发平伏** | 竖毛肌 **舒张**，毛发平贴在皮肤上；这减少了紧贴皮肤的绝缘空气层，使更多热量得以散发。 |

**当体温下降时（过冷）：**

| 反应 | 机制 |
| :--- | :--- |
| **战栗** | 骨骼肌 **快速而不自主地** 收缩和舒张。这种收缩需要呼吸作用，而呼吸作用释放出热作为副产物，使身体变暖。 |
| **血管收缩** | 供皮肤表面毛细血管的小动脉 **收缩**（变窄），将血液从皮肤表面分流。从血液散失到周围环境的热量减少。 |
| **毛发竖起（竖毛反射）** | 竖毛肌 **收缩**，将毛发拉直。这会在皮肤表面捕集一层较厚的绝缘 **空气**，减少热量散失。（对人类效果有限，但属于退化反射。） |
| **隔热** | 皮下组织中的 **脂肪组织** 层提供隔热保温作用，减少体核的热量散失。 |

额外的保暖机制包括代谢率增加（由肾上腺素和甲状腺素刺激），以及在极冷环境下的有意识行为，如穿额外衣物和寻找庇护所。

### 14.5 向性反应

#### 14.5.1 什么是向性反应？

**向性** 是植物的一种定向生长反应，在该反应中，刺激的方向决定了生长的方向。向性由植物激素（化学信号）控制，主要是 **生长素**。

#### 14.5.2 向光性

**向光性** 是植物部分朝着或背离光线方向生长的反应。

| 植物部分 | 反应 | 类型 |
| :--- | :--- | :--- |
| **茎** | **朝向光** 生长 | **正向光性** |
| **根** | **背离光** 生长 | **负向光性** |

**向光性为何重要？** 茎弯向光以最大限度地吸收光用于 **光合作用**，使植物能产生更多葡萄糖并更高效地生长。

#### 14.5.3 向地性

**向地性**（也称向重力性）是植物部分朝着或背离重力方向生长的反应。

| 植物部分 | 反应 | 类型 | 目的 |
| :--- | :--- | :--- | :--- |
| **根** | **朝向重力** 生长 | **正向地性** | 固定植物，并从土壤中吸收水分和矿物质 |
| **茎** | **背离重力** 生长 | **负向地性** | 伸出土壤接受光照进行光合作用 |

#### 14.5.4 生长素在控制茎生长中的作用（补充内容）

**生长素** 是一种通过刺激 **细胞伸长** 来控制生长的植物激素。必须记住以下几点：

1. **生长素在茎尖产生**（顶端分生组织）。
2. **生长素通过扩散** 从茎尖沿茎向下运输。
3. **生长素分布不均**，在光（向光性）和重力（向地性）的作用下，会移动到 **背光侧** 或 **近地侧**。
4. **生长素刺激细胞伸长**——生长素浓度较高一侧的细胞比浓度较低一侧的细胞伸长（长得更长）得更多。

**用生长素解释茎的向光性：**
- 当光照射到茎的一侧时，茎尖产生的生长素优先移动到 **背光（较暗）的一侧**。
- 背光侧较高的生长素浓度导致该侧的细胞比向光侧的细胞 **伸长更多**。
- 这种差异生长导致茎 **向光弯曲**。

**用生长素解释茎的向地性：**
- 当茎被水平放置时，重力使生长素积聚在茎的 **近地侧**。
- 近地侧较高的生长素浓度刺激该侧细胞更快地伸长。
- 这导致茎 **向上弯曲**，背离重力。

**关于根的说明**：在根中，对生长素的反应不同——高浓度的生长素会 **抑制** 根细胞的伸长。这就是为什么根会 **朝向** 重力弯曲（近地侧生长素浓度高 → 生长受抑制 → 背地侧生长更快 → 根向下弯曲）。大纲着重于茎的反应；了解根的情况有助于提供背景信息。

## 第 3 部分：词汇表（教学大纲要求的关键术语）

| 术语 | 定义 |
| :--- | :--- |
| **刺激 (Stimulus)** | 内、外环境中能被感受器探测到的变化。 |
| **感受器 (Receptor)** | 探测刺激的特化细胞或细胞群。 |
| **效应器 (Effector)** | 对刺激产生反应的肌肉或腺体。 |
| **神经元 (Neurone)** | 神经细胞；一种传导电信号的特化细胞。 |
| **中枢神经系统 (CNS)** | 脑和脊髓；处理信息并协调反应。 |
| **周围神经系统 (PNS)** | 脑和脊髓以外的所有神经；将冲动传入和传出中枢神经系统。 |
| **反射弧 (Reflex arc)** | 参与反射动作的神经元通路：感受器 → 感觉神经元 → 中间神经元 → 运动神经元 → 效应器。 |
| **反射动作 (Reflex action)** | 对刺激的一种自动、快速、不随意的反应，可保护身体免受伤害。 |
| **突触 (Synapse)** | 两个神经元之间的连接点，此处释放神经递质以传递冲动。 |
| **神经递质 (Neurotransmitter)** | 在突触处释放的化学信使；扩散通过突触间隙，并与下一个神经元上的受体蛋白结合。 |
| **感觉器官 (Sense organ)** | 对特定刺激作出反应的受体细胞群（例如眼、耳、皮肤）。 |
| **瞳孔反射 (Pupil reflex)** | 瞳孔大小随光强自动调节，由虹膜内的拮抗肌控制。 |
| **调节 (Accommodation)** | 晶状体改变形状以聚焦于近处或远处物体的过程。 |
| **视杆细胞 (Rods)** | 视网膜中对弱光敏感的感光细胞；探测光强度（黑白视觉）。 |
| **视锥细胞 (Cones)** | 集中在中央凹的感光细胞；探测颜色（三种类型：红、绿、蓝）。 |
| **中央凹 (Fovea)** | 视网膜中心的一个凹陷，视锥细胞密度极高；是色觉最清晰的区域。 |
| **激素 (Hormone)** | 由内分泌腺产生的化学物质，由血液运输，能改变一个或多个特定靶器官的活动。 |
| **内分泌腺 (Endocrine gland)** | 将激素直接分泌到血液中的腺体。 |
| **肾上腺素 (Adrenaline)** | 由肾上腺分泌的“战斗或逃跑”激素；增加心率、呼吸频率、瞳孔直径和血糖浓度。 |
| **胰岛素 (Insulin)** | 由胰腺分泌的一种激素，通过刺激肝脏将葡萄糖转化为糖原来降低血糖浓度。 |
| **胰高血糖素 (Glucagon)** | 由胰腺分泌的一种激素，通过刺激肝脏将糖原分解为葡萄糖来升高血糖浓度。 |
| **稳态 (Homeostasis)** | 身体内部环境维持恒定的状态。 |
| **负反馈 (Negative feedback)** | 一种稳态机制，当偏离设定点时触发反应，以逆转该变化，恢复正常状态。 |
| **血管舒张 (Vasodilation)** | 供皮肤表面毛细血管的小动脉舒张（变宽），增加靠近皮肤表面的血流量以散热。 |
| **血管收缩 (Vasoconstriction)** | 供皮肤表面毛细血管的小动脉收缩（变窄），减少靠近皮肤表面的血流量以保存热量。 |
| **向性 (Tropism)** | 植物的一种定向生长反应，方向由刺激的方向决定。 |
| **向光性 (Phototropism)** | 植物部分朝向或背离光生长的反应。 |
| **向地性 (Gravitropism / Geotropism)** | 植物部分朝向或背离重力方向生长的反应。 |
| **生长素 (Auxin)** | 一种在茎尖产生的植物激素；通过植物扩散；响应光和重力而不均匀分布；刺激细胞伸长。 |

## 第 4 部分：常见考试错误与纠正

| 常见错误 | 正确答案 |
| :--- | :--- |
| 混淆瞳孔和虹膜 | **虹膜** 是彩色的肌肉环；**瞳孔** 是中央让光通过的孔洞 |
| 写“眼睛调节时晶状体移动” | 晶状体 **改变形状**（视近物时变厚，视远物时变薄）；它并不移动 |
| 混淆血管舒张和血管收缩 | **血管舒张**：小动脉变宽 → 更多血液靠近皮肤表面 → 散热（热时）。**血管收缩**：小动脉变窄 → 更少血液靠近皮肤表面 → 保温（冷时） |
| 说“大脑控制所有反射” | 反射弧经过 **脊髓**（或脑干）；大脑只在事后才获知 |
| 说“激素传播到特定器官” | 激素随血液流遍 **全身**，但仅影响具有 **互补受体** 的靶器官 |
| 混淆视杆细胞和视锥细胞 | **视杆细胞** = 弱光、黑/白、遍布视网膜。**视锥细胞** = 强光、彩色、集中于中央凹 |
| 写“睫状肌收缩以聚焦远处物体” | 事实恰恰相反：视远物时睫状肌 **舒张**，视近物时 **收缩** |
| 混淆生长素在茎和根中的作用 | 在 **茎** 中，生长素 **促进** 生长（细胞伸长更多）。在 **根** 中，生长素 **抑制** 生长 |
| 说“胰岛素升高血糖” | **胰岛素降低** 血糖；**胰高血糖素升高** 血糖 |
| 将反射称为“随意动作” | 反射是 **不随意** 且 **自动** 的 |
