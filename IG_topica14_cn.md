### 📌 一、Paper 2 – 高频考点与答题关键词 (选择题)

| 考点类别 | 真题关键词 (Question Keyword) | 答题关键词 (Answer Key) |
| :--- | :--- | :--- |
| **中枢神经系统组成** | Which parts make up the central nervous system (CNS)? | brain and spinal cord |
| **神经元功能** | Which neurone carries impulses from receptors to the CNS? | sensory neurone |
| **反射弧终点** | What is the effector in a reflex arc? | muscle or gland |
| **突触方向性** | Why can impulses only travel in one direction across a synapse? | neurotransmitter vesicles only on one side / receptors only on the other side |
| **瞳孔反射（强光）** | What happens to the pupil in bright light? | constricts (gets smaller) |
| **瞳孔反射（弱光）** | Which muscle contracts in dim light? | radial muscle |
| **看近物时睫状肌状态** | What happens to ciliary muscles when focusing on a near object? | contract |
| **看远物时悬韧带状态** | What happens to suspensory ligaments when focusing on a distant object? | tighten |
| **激素运输方式** | How are hormones transported around the body? | in the blood |
| **肾上腺素作用** | Which hormone increases heart rate during ‘fight or flight’? | adrenaline |
| **胰岛素作用** | What effect does insulin have on blood glucose? | decreases (lowers) blood glucose |
| **负反馈原理** | What is negative feedback? | a process that reverses a change to maintain a set point |
| **植物向光性** | What is phototropism? | growth of a plant towards or away from light |
| **体温调节（热时血管）** | What happens to skin arterioles when the body is too hot? | vasodilation (widen) |
| **体温调节（冷时肌肉）** | What generates heat when the body is too cold? | shivering |

### 📝 二、Paper 4 – 典型题目与答题模板 (理论题)

| 题型 | 真题关键词 | 答题模板 (按评分标准) |
| :--- | :--- | :--- |
| **反射弧路径** | Describe the path of a simple reflex arc. | Stimulus → receptor → sensory neurone → relay neurone (in CNS) → motor neurone → effector → response |
| **突触传递过程 (Supplement)** | Describe the events that occur at a synapse. | (1) Electrical impulse reaches synaptic knob → (2) vesicles release neurotransmitter into synaptic gap → (3) neurotransmitter diffuses across gap → (4) binds to receptors on next neurone → (5) triggers new electrical impulse |
| **瞳孔反射机制 (Supplement)** | Explain how the pupil reflex works in bright light. | In bright light, circular muscles contract, radial muscles relax → pupil constricts (gets smaller) → less light enters the eye |
| **调节作用 (Accommodation)** | Explain how the eye focuses on a near object. | Ciliary muscles contract → suspensory ligaments slacken → lens becomes fatter / more curved → light refracted more strongly |
| **视杆细胞与视锥细胞区别 (Supplement)** | Describe the distribution and function of rods and cones. | Rods: found throughout retina (except fovea), sensitive in dim light (night vision). Cones: concentrated in fovea, require bright light, provide colour vision |
| **肾上腺素作用** | Describe the effects of adrenaline during ‘fight or flight’. | Increases heart rate, breathing rate, pupil diameter, and blood glucose concentration |
| **血糖调节** | Explain how the body lowers high blood glucose concentration. | Pancreas secretes insulin → insulin causes liver and muscles to take up glucose and convert it to glycogen for storage → blood glucose decreases |
| **体温调节机制** | Describe how the body cools down when too hot. | Vasodilation: arterioles widen → more blood flows near skin surface → heat lost by radiation. Sweating increases → sweat evaporates, taking heat from the body. Hair erector muscles relax. |
| **植物向光性解释 (Supplement)** | Explain phototropism in a shoot. | Auxin is produced in shoot tip. Light causes auxin to accumulate on shaded side → auxin stimulates cell elongation on shaded side → shaded side grows faster → shoot bends towards light |
| **植物向地性解释 (Supplement)** | Explain gravitropism in a root. | Auxin accumulates on lower side of root → auxin inhibits cell elongation on lower side → upper side grows faster → root bends downwards (towards gravity) |

### ⚠️ 三、Paper 2 常见陷阱（基于真题失分点）

| 陷阱 | 正确理解 | 错误选项特征 |
| :--- | :--- | :--- |
| **混淆反射弧路径顺序** | receptor → sensory → relay → motor → effector | 选项中顺序颠倒或遗漏 relay neurone |
| **混淆神经元功能** | sensory = 传入 CNS；motor = 传出 CNS 到 effector | 选项中把 sensory 写成从 CNS 传出 |
| **混淆瞳孔反射肌肉作用** | 强光：circular 收缩；弱光：radial 收缩 | 选项中写反两种肌肉的收缩/放松 |
| **混淆调节作用 (Accommodation)** | 看近物：睫状肌**收缩**，悬韧带**放松** | 选项中写“ciliary muscles relax” |
| **激素 vs 神经调节速度** | nervous: very fast; hormonal: relatively slow | 选项中写“hormones are faster than nerves” |
| **混淆胰岛素和胰高血糖素** | insulin ↓ 血糖；glucagon ↑ 血糖 | 选项中交换两者的作用 |
| **混淆植物向光性与向地性** | shoot: 向光性（+），向地性（-）；root: 向地性（+） | 选项中写“root grows away from gravity” |
| **混淆恒温与变温动物** | mammals and birds are homeotherms (恒温) | 选项中写“fish maintain constant body temperature” |
| **遗忘“效应器”定义** | effector = muscle or gland that produces response | 选项中只写“muscle”，遗漏“gland” |
| **混淆激素与神经递质** | hormones: 通过血液运输；neurotransmitters: 在突触间隙扩散 | 选项中写“hormones are released at synapses” |

### 📌 四、单点考题示例 (Single Examination Point Questions)

| 单点考题 | 答案 |
| :--- | :--- |
| Name the two parts of the central nervous system. | Brain and spinal cord. |
| State the function of a sensory neurone. | Carries impulses from receptors to the CNS. |
| What is a reflex action? | An automatic, rapid response to a stimulus that does not involve conscious thought. |
| Name the gap between two neurones. | Synapse / synaptic cleft. |
| State the function of the cornea. | Refracts (bends) light into the eye. |
| Which part of the eye contains light receptors? | Retina. |
| Name the hormone that increases heart rate during danger. | Adrenaline. |
| Which gland produces insulin? | Pancreas. |
| What is homeostasis? | Maintenance of a constant internal environment. |
| What is the name of the plant hormone that controls tropisms? | Auxin. |
| Which part of the brain detects changes in body temperature? | Hypothalamus. |
| State two responses of the body when it is too hot. | Vasodilation; sweating. |
| What is the name of the muscle that contracts to make the pupil smaller? | Circular muscle. |
| Describe what happens to the lens when focusing on a distant object. | Lens becomes thinner / flatter. |
