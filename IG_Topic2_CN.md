## 🔬 Topic 2: Organisation of the Organism

本资料基于Cambridge IGCSE Biology (0610/0970) 2024–2025考纲及历年真题整理。  
所有答案均按 **CAIE官方评分标准 (Mark Scheme)** 提炼，可直接用于备考。

### 📌 一、Paper 2 – 高频考点与答题关键词 (选择题)

| 考点类别 | 真题关键词 (Question Keyword) | 答题关键词 (Answer Key) |
| :--- | :--- | :--- |
| **植物细胞 vs 动物细胞** | Which structure is found in plant cells but not in animal cells? | plant only: cell wall, chloroplast, large permanent vacuole[reference:0] |
| **细胞器功能** | State the function of the mitochondria / nucleus / ribosomes | mitochondria: site of aerobic respiration / releases energy; nucleus: contains DNA, controls cell activities; ribosomes: protein synthesis[reference:1] |
| **细菌细胞特征** | Which structure is found in a bacterial cell but not in a plant or animal cell? | plasmids, circular DNA (not in a nucleus), cell wall (not made of cellulose)[reference:2] |
| **放大倍率计算** | Calculate the magnification / actual size of the specimen | Magnification = image size ÷ actual size (must be in same units)[reference:3] |
| **单位换算** | Convert mm to μm / μm to mm | 1 mm = 1000 μm (multiply mm → μm; divide μm → mm) |
| **组织层次** | Place in order from smallest to largest / largest to smallest | cell → tissue → organ → organ system → organism[reference:4] |
| **组织识别** | Identify the level of organisation: phloem / small intestine / leaf / heart | phloem: tissue; small intestine / leaf / heart: organ |
| **干细胞定义** | What is a stem cell? | undifferentiated / unspecialised cell that can divide by mitosis to produce daughter cells that can become specialised for specific functions[reference:5] |
| **红细胞适应** | How is a red blood cell adapted to carry oxygen? | biconcave shape (large surface area); no nucleus (more space for haemoglobin); contains haemoglobin (binds oxygen)[reference:6] |
| **植物与细菌共有结构** | State one structure present in both plant cells and bacterial cells | cell wall (plant cell wall is cellulose; bacterial cell wall is not cellulose – but both have a cell wall)[reference:7] |

### 📝 二、Paper 4 – 典型题目与答题模板 (理论题)

| 题型 | 真题关键词 | 答题模板 (按评分标准) |
| :--- | :--- | :--- |
| **细胞器功能** | State the function of the mitochondria / nucleus / ribosomes | mitochondria: site of aerobic respiration / releases energy for the cell; nucleus: contains DNA / genetic material, controls cell activities; ribosomes: site of protein synthesis |
| **植物 vs 动物细胞** | State two structures present in plant cells but absent in animal cells | cell wall, chloroplast, large permanent vacuole (any two) |
| **细菌细胞结构** | Describe the structure of a bacterial cell | cell wall, cell membrane, cytoplasm, ribosomes, circular DNA (free in cytoplasm, not enclosed in a nucleus), plasmids (small circles of DNA) |
| **组织层次定义** | Define the term tissue / organ / organ system | tissue: a group of cells with similar structures working together to perform a shared function[reference:8]; organ: a structure made up of a group of different tissues working together to perform a specific function[reference:9]; organ system: a group of different organs working together to perform a major body function |
| **放大倍率计算** | Calculate the magnification of the drawing / photograph | Magnification = image size ÷ actual size (both in same units); show working; include units in final answer |
| **单位换算** | Convert measurements from mm to μm / μm to mm | 1 mm = 1000 μm; to convert mm to μm, multiply by 1000; to convert μm to mm, divide by 1000 |
| **特化细胞适应** | Describe how a red blood cell / root hair cell / palisade cell is adapted for its function | red blood cell: biconcave shape (large surface area), no nucleus (more space for haemoglobin), contains haemoglobin (binds oxygen); root hair cell: long extension (large surface area for absorption); palisade cell: many chloroplasts (absorb more light for photosynthesis)[reference:10] |
| **细胞分化** | Explain what is meant by cell differentiation | the process by which a cell becomes specialised to carry out a specific function; cells develop different structures and characteristics to perform particular jobs |
| **干细胞应用** | Explain why stem cells are important in medicine | can differentiate into many different cell types; can be used to repair damaged tissues / treat diseases (e.g. leukaemia, Parkinson’s, spinal cord injury) |
| **新细胞来源** | State how new cells are produced | by division of existing cells (mitosis) |
| **放大倍率计算（步骤）** | Explain why units must be converted before magnification calculation | to ensure that image size and actual size are in the same units before division, otherwise the magnification calculated will be incorrect |

### ⚠️ 三、Paper 2 常见陷阱（基于真题失分点）

| 陷阱 | 正确理解 | 错误选项特征 |
| :--- | :--- | :--- |
| 细菌有真正的细胞核 | 细菌是原核生物，没有真正的细胞核，DNA是环状的游离在细胞质中[reference:11] | 选项中写"has a nucleus" |
| 植物细胞有中心粒 | 中心粒只存在于动物细胞，植物细胞没有 | 选项中把中心粒放在植物细胞中 |
| 红细胞有细胞核 | 成熟红细胞没有细胞核（为了给血红蛋白腾出空间） | 选项中写红细胞有细胞核[reference:12] |
| 线粒体功能是光合作用 | 线粒体是有氧呼吸的场所，叶绿体才是光合作用的场所 | 选项中把线粒体功能写成"photosynthesis"[reference:13] |
| 放大倍率公式用错 | Magnification = image size ÷ actual size (不是 actual size ÷ image size)[reference:14] | 选项中把公式倒过来写 |
| 组织层次顺序颠倒 | 从小到大：cell → tissue → organ → organ system → organism[reference:15] | 选项中把 organ 放在 tissue 前面 |
| 干细胞只能变成一种细胞 | 干细胞可以分化成多种特化细胞，但胚胎干细胞比成体干细胞分化能力更强[reference:16] | 选项中写"stem cells can only become one type of cell" |
| 单位换算忘记乘除1000 | 1 mm = 1000 μm，换算时乘除方向错误 | 选项中写"1 mm = 100 μm" |
| 植物细胞和细菌细胞壁相同 | 植物细胞壁由纤维素组成，细菌细胞壁由肽聚糖组成，但两者都有细胞壁 | 选项中写"bacterial cell wall is made of cellulose"[reference:17] |
| 细胞膜是植物细胞特有 | 细胞膜是所有细胞（植物、动物、细菌）都有的结构 | 选项中写"cell membrane is only found in animal cells" |
