## 4.1 Elemental Composition of Biological Molecules

| Molecule type | Elements present | Notes |
|:---|:---|:---|
| **Carbohydrates** | C, H, O | H:O = 2:1 (general formula Cₙ(H₂O)ₙ) |
| **Fats (lipids)** | C, H, O | Higher H:O ratio than carbohydrates |
| **Proteins** | C, H, O, N, sometimes S | S found in sulfur‑containing amino acids (e.g. cysteine, methionine) |
| **Vitamin C** | C, H, O | Ascorbic acid |
| **Water** | H, O | Inorganic but essential solvent |
| **DNA** | C, H, O, N, P | Phosphate group contains P; deoxyribose contains C,H,O; bases contain N |

## 4.2 Food Tests (Common Chemical Tests)

### 4.2.1 Starch test

| Item | Details |
|:---|:---|
| **Reagent** | Iodine solution |
| **Positive result** | Yellow/brown → **blue‑black** |
| **Negative result** | Remains yellow/brown |
| **Note** | After starch digestion by amylase, iodine stays yellow/brown |

### 4.2.2 Reducing sugar test (Benedict’s test)

| Item | Details |
|:---|:---|
| **Reagent** | Benedict's solution |
| **Conditions** | Heat (boiling water bath for 2–5 minutes) |
| **Positive result** | Blue → green (trace) → yellow → orange → **brick‑red precipitate** (large amount) |
| **Note** | Non‑reducing sugars (e.g. sucrose) must be hydrolysed first |

### 4.2.3 Protein test (Biuret test)

| Item | Details |
|:---|:---|
| **Reagent** | Biuret reagent |
| **Positive result** | Blue → **violet (purple)** |
| **Negative result** | Remains blue |
| **Note** | Alkaline conditions needed: add NaOH first, then a few drops of dilute CuSO₄ |

### 4.2.4 Fat test (ethanol emulsion test)

| Item | Details |
|:---|:---|
| **Reagent** | Ethanol + water |
| **Procedure** | 1. Add about 2 cm³ ethanol to sample, shake well. 2. Add equal volume of water, shake again. |
| **Positive result** | **Milky‑white emulsion** |
| **Negative result** | Clear solution |
| **Note** | For solid samples, grind first before adding ethanol. |

### 4.2.5 Vitamin C test (DCPIP test)

| Item | Details |
|:---|:---|
| **Reagent** | DCPIP (blue solution) |
| **Principle** | Vitamin C reduces DCPIP (turns it colourless) |
| **Procedure** | 1. Add a fixed volume of DCPIP. 2. Add sample drop by drop, shaking after each drop. 3. Record number of drops needed to decolourise DCPIP. |
| **Positive result** | Blue → colourless |
| **Semi‑quantitative use** | Fewer drops needed → higher vitamin C concentration (compare with standard) |
| **Note** | Fresh samples required; vitamin C destroyed by heat, light, oxygen. |

## 4.3 Carbohydrates

### 4.3.1 Classification (high‑frequency exam topic)

| Type | Examples | Features | Exam points |
|:---|:---|:---|:---|
| **Monosaccharides** | glucose, fructose, galactose | simple sugars, reducing sugars, soluble | used in respiration; building blocks for di‑ and polysaccharides |
| **Disaccharides** | maltose (glucose + glucose) | product of starch digestion by amylase | |
| | sucrose (glucose + fructose) | table sugar; transported in plant phloem | |
| | lactose (glucose + galactose) | sugar in mammalian milk | |
| **Polysaccharides** | starch | glucose polymer (plants) | turns iodine blue‑black; plant energy store |
| | glycogen | glucose polymer (animals) | animal energy store; highly branched |
| | cellulose | glucose polymer (β‑1,4 links) | **main component of plant cell walls**; humans cannot digest it (no cellulase) |

> ⭐ **Exam common questions**:  
> - “Polysaccharide for energy storage in plants” → **starch**; in animals → **glycogen**.  
> - “Polysaccharide that makes up plant cell walls” → **cellulose**.  
> - “Give an example of a disaccharide” → sucrose, maltose or lactose.

### 4.3.2 Related equations and roles

- Glucose test: reducing sugar + Benedict’s → brick‑red precipitate.
- Starch digestion: starch –(amylase)→ maltose –(maltase)→ glucose.
- Cellulose as dietary fibre: promotes peristalsis, prevents constipation.

## 4.4 Fats (Lipids)

- **Composition**: 1 glycerol + 3 fatty acids → triglyceride.
- **Saturated vs unsaturated** (supplementary knowledge):
  - Saturated: no C=C double bonds, solid at room temperature (animal fats).
  - Unsaturated: contain C=C double bonds, liquid at room temperature (vegetable oils).
- **Functions**:
  - Energy store (more than twice the energy per gram compared to carbohydrates)
  - Insulation (subcutaneous fat, whale blubber)
  - Protection of internal organs (cushioning)
  - Component of cell membranes (phospholipids)
  - Solvent for fat‑soluble vitamins (A, D, E, K)
- **Exam question**: Why are fats better for long‑term energy storage than carbohydrates?  
  → High energy density, insoluble in water (does not affect osmotic balance), large storage capacity.

## 4.5 Proteins

- **Building blocks**: amino acids joined by peptide bonds → polypeptide chains.
- **Levels of structure** (brief): primary (amino acid sequence) → secondary (α‑helix, β‑sheet) → tertiary (3D conformation) → quaternary (multiple chains, e.g. haemoglobin).
- **Functions (give examples)**:
  - Enzymes (catalysis)
  - Hormones (e.g. insulin)
  - Antibodies (immunity)
  - Structural proteins (collagen, keratin)
  - Transport proteins (haemoglobin)
- **Test**: Biuret test → violet.
- **Nitrogen balance**: proteins contain nitrogen; breakdown produces nitrogenous waste (urea) → deamination in liver, excretion by kidneys.

## 4.6 Vitamin C

- **Chemical name**: ascorbic acid.
- **Sources**: fresh fruits (citrus, kiwi), vegetables (peppers, broccoli).
- **Functions**: antioxidant, collagen synthesis, enhances iron absorption, immunity.
- **Deficiency disease**: scurvy – bleeding gums, slow wound healing.
- **Test**: DCPIP decolourisation (semi‑quantitative).

## 4.7 Water

- **Importance**:
  - Excellent solvent (most biochemical reactions occur in solution)
  - Transport medium (blood, lymph, plant sap)
  - Participates in chemical reactions (hydrolysis, photosynthesis)
  - Temperature regulation (high specific heat capacity → buffers temperature; evaporative cooling)
  - Lubricant (synovial fluid, mucus)
- **Exam point**: Why is water essential as a solvent? → Polar molecule; dissolves ions and polar substances; non‑polar molecules (e.g. fats) do not dissolve.

## 4.8 DNA

### 4.8.1 Structure and composition

- **Full name**: Deoxyribonucleic acid.
- **Building blocks**: nucleotides – each nucleotide = phosphate + deoxyribose sugar + nitrogenous base.
- **Bases**:
  - Purines: adenine (A), guanine (G)
  - Pyrimidines: thymine (T), cytosine (C)
- **Double helix** (Watson & Crick):
  - Two antiparallel strands held together by hydrogen bonds between bases: **A = T** (two bonds), **G ≡ C** (three bonds).
  - Sugar‑phosphate backbone on the outside, bases inside.
- **Function**: carries genetic information, controls protein synthesis (via transcription and translation).

### 4.8.2 Link to protein synthesis (brief)

- DNA transcribed into mRNA; mRNA translated into polypeptide chain.
- Every three bases (codon) codes for one amino acid.
- (Detailed mechanism belongs to Topic 17 – Inheritance.)

## 📌 Common Exam Traps (Paper 2)

| Wrong idea | Correct statement | Sub‑topic |
|:---|:---|:---|
| All carbohydrates turn iodine blue‑black | Only starch gives blue‑black; glucose, sucrose etc. do not | 4.3 |
| All sugars give a positive Benedict’s test | Non‑reducing sugars (e.g. sucrose) must be hydrolysed first | 4.2 |
| Biuret test detects fats | Biuret is for proteins; fat test is the ethanol emulsion test | 4.2 |
| Vitamin C decolourises iodine | Vitamin C decolourises **DCPIP**, not iodine | 4.2 |
| Cellulose is a type of starch | Cellulose is also a glucose polymer, but different structure; humans cannot digest it | 4.3 |
| Glycogen is the plant storage carbohydrate | Plants store starch; animals store glycogen | 4.3 |
| Fats are only for energy storage | Fats also provide insulation, protection, cell membrane component, etc. | 4.4 |
| Proteins contain only C, H, O | Proteins always contain N, and sometimes S | 4.1 |
| DNA contains ribose | DNA contains **deoxyribose**; RNA contains ribose | 4.8 |
