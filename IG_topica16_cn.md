## 第 2 部分：详细知识点

### 16.1 无性生殖

#### 16.1.1 什么是无性生殖？（核心内容）

无性生殖是一个亲本产生 **基因完全相同** 的后代（克隆）的过程。它不涉及配子（性细胞）或受精。细胞分裂通过 **有丝分裂** 进行，产生与亲本细胞遗传物质完全相同的副本。后代的基因与亲本完全一样——它们是 **克隆体**。

无性生殖的关键特征：
- 仅涉及 **一个亲本**
- 没有配子的融合
- 没有遗传信息的混合
- 后代与亲本之间以及后代彼此之间 **基因完全相同**
- 细胞分裂通过 **有丝分裂** 进行

#### 16.1.2 无性生殖的实例（核心内容）

| 生物 | 无性生殖方式 | 描述 |
| :--- | :--- | :--- |
| **细菌** | 二分裂 | 细菌细胞的单一环状 DNA 复制；细胞随后分裂成两个基因完全相同的子细胞。速度极快——在理想条件下，细菌每 20 分钟可以繁殖一倍。 |
| **酵母菌（真菌）** | 出芽 | 从亲本细胞侧面长出一个小芽。随着芽体增大，亲本的细胞核分裂，一个子细胞核移入芽体。芽体最终脱落，形成一个基因相同的新酵母细胞。 |
| **水螅** | 出芽 | 从亲本体壁长出一个小型水螅芽体。发育到一定程度后，它脱离亲本独立生活。 |
| **草莓** | 匍匐茎（长条枝） | 地上水平茎从亲本植株长出。在匍匐茎的节上，长出根和新芽，形成与亲本基因相同的新植株。最终匍匐茎退化，后代独立。 |
| **马铃薯** | 块茎 | 地下茎膨大形成块茎。每个块茎储存淀粉并带有芽眼。每个芽眼都可以萌发并长成一株新的、基因相同的马铃薯植株。 |
| **洋葱/大蒜** | 鳞茎 | 鳞茎是一个短的地下茎，周围由充满储存养料的肥大叶基包裹。新鳞茎从母鳞茎基部长出，每一个都能产生一株基因相同的植株。 |

#### 16.1.3 无性生殖的优缺点（补充内容）

**（一）对野生种群而言**：

| 优点 | 缺点 |
| :--- | :--- |
| 仅需一个亲本——无需寻找配偶 | 无遗传变异——所有后代都是克隆体，完全一样 |
| 能够快速高效地繁殖 | 所有个体对相同的疾病或环境变化同样脆弱 |
| 迅速占据适宜的生境——一个个体就能发展成一个种群 | 无法适应新的或变化的条件——如果环境发生不利变化，整个种群可能被摧毁 |
| 成功性状原封不动地遗传下去 | 单一有害突变对所有后代影响相同 |

**（二）对作物生产（农业）而言**：

| 优点 | 缺点 |
| :--- | :--- |
| 农民可以种植具有理想性状（如口感、大小、产量）且基因相同的植株 | 作物基因完全相同——对新型病害毫无抵抗力 |
| 品质和外观整齐划一——便于收割、分级和销售 | 过度依赖单一品种，一旦病害来袭就有整片绝收的风险 |
| 高产品种繁殖速度更快 | 无法通过自然变异发展出新性状 |
| 对繁殖无籽水果有用（如香蕉、无籽葡萄） | 降低农业系统的生物多样性 |

### 16.2 有性生殖

#### 16.2.1 什么是有性生殖？（核心内容）

有性生殖是 **两个配子**（性细胞）的核 **融合** 形成 **受精卵**，并产生彼此 **基因不同** 的后代的过程。

- **受精** 定义为配子核的融合。
- 有性生殖需要 **两个亲本**。
- 配子由 **减数分裂** 产生——这是一种染色体数目减半的细胞分裂方式。
- 由于减数分裂产生的配子具有不同的等位基因组合，因此后代彼此不同，也与任一亲本不同。
- 遗传变异来自：(1) 减数分裂过程中染色体的随机分配，(2) 同源染色体之间的交叉互换，(3) 随机交配，(4) 随机受精（哪个精子使哪个卵子受精）。

#### 16.2.2 单倍体与二倍体细胞核（补充内容）

- **单倍体细胞核** 是只含 **一套** 染色体的核。人类单倍体细胞含有 23 条染色体。**配子**（精子和卵细胞）是单倍体。
- **二倍体细胞核** 是含有 **两套** 染色体的核。人类二倍体细胞含有 46 条染色体（23 对）。受精形成的 **受精卵** 是二倍体——它从双亲各获得一套染色体，恢复了完整的染色体数目。

#### 16.2.3 有性生殖的优缺点（补充内容）

**（一）对野生种群而言**：

| 优点 | 缺点 |
| :--- | :--- |
| 在种群内产生 **遗传变异**——后代各不相同 | 需要两个亲本——必须找到配偶 |
| 遗传变异为 **自然选择** 和对环境变化的适应提供了原材料 | 比无性生殖慢 |
| 如果疾病来袭或环境改变，某些个体更有可能存活 | 生长和发育耗时更长 |
| 防止有害隐性突变的积累 | 耗能多（需要减数分裂、产生配子、交配等） |

**（二）对作物生产而言**：

| 优点 | 缺点 |
| :--- | :--- |
| 遗传变异允许通过选择性育种培育具有优良性状（抗病、高产、优质）的新品种 | 后代与亲本不完全相同——无法保证整齐一致的品质 |
| 种群可以适应不断演化的病虫害 | 比无性繁殖需要更多时间和资源 |
| 更大的遗传多样性提供应对环境变化的韧性 | 由于随机的遗传重组，某些后代可能出现不良性状 |

### 16.3 植物的有性生殖

#### 16.3.1 虫媒花的结构（核心内容）

你必须能够在图表和图像中 **识别并绘制** 以下结构。

**雄性生殖部分（雄蕊）：**

| 结构 | 功能 |
| :--- | :--- |
| **花丝** | 支撑并托起花药，将其置于来访昆虫会碰到的位置 |
| **花药** | 产生并含有 **花粉粒**（雄配子）。花粉成熟时花药裂开（开裂）以释放花粉粒 |

**雌性生殖部分（心皮/雌蕊）：**

| 结构 | 功能 |
| :--- | :--- |
| **柱头** | 心皮顶端有粘性的表面，在传粉过程中 **捕获并黏住花粉粒**。其粘性通常来自含糖的分泌物 |
| **花柱** | 连接柱头和子房的柄。它支撑着柱头，**花粉管** 从柱头上萌发的花粉粒长出后 **向下穿过它** 生长 |
| **子房** | 心皮基部中空的部分，内含 **胚珠**。受精后，子房发育为 **果实** |
| **胚珠** | 位于子房内部。每个胚珠含有一个雌配子（卵细胞核）。受精后，每个胚珠发育为一粒 **种子** |

**其他结构：**

| 结构 | 功能 |
| :--- | :--- |
| **萼片** | 绿色的叶状结构，在花蕾开放前 **保护发育中的花蕾** |
| **花瓣** | 通常 **颜色鲜艳**，有时 **有香味**，以吸引昆虫传粉者。它们还可能有 **蜜腺引导图案**（昆虫可见的图案），引导传粉者找到花蜜和生殖结构 |
| **蜜腺** | 子房或花瓣基部的腺体，产生 **花蜜**——一种含糖液体，作为对来访昆虫传粉的回馈 |

**图示参考**：雄性部分为 **雄蕊**（= 花丝 + 花药）。雌性部分为 **心皮/雌蕊**（= 柱头 + 花柱 + 子房 + 胚珠）。

#### 16.3.2 风媒花（核心内容）

风媒花看起来与虫媒花非常不同。你必须能够识别并描述风媒花的 **花药和柱头**。

**风媒花的特征：**

| 特征 | 描述 |
| :--- | :--- |
| **花瓣** | 小，颜色暗淡（常为绿色或褐色），或缺失——无需吸引昆虫 |
| **花蜜和气味** | 缺失——不提供任何回报 |
| **花药** | 大，连接松散，悬挂在 **花外** 的长而灵活的花丝上，以便风能轻易摇散花粉 |
| **柱头** | 大，呈 **羽毛状或刷状**，悬挂在 **花外**，提供大面积来捕获空气中风媒传播的花粉 |
| **花粉** | 产量 **极大**，因为大部分被浪费（由随机的风传播，而非由昆虫定向携带）。花粉粒 **小、轻、光滑且干燥**，以便能被风轻易携带 |

#### 16.3.3 虫媒花与风媒花的对比（核心内容）

| 特征 | 虫媒花 | 风媒花 |
| :--- | :--- | :--- |
| **花瓣** | 大，颜色鲜艳，常有香味，有时有蜜腺引导图案 | 小，绿色/褐色，暗淡，有时缺失 |
| **花蜜** | 有——作为对昆虫的回报 | 无 |
| **气味** | 常有 | 无 |
| :--- | :--- | :--- |
| **花药** | 附着牢固，位于花内昆虫会碰到的位置 | 大，连接松散，悬挂在花外长而灵活的花丝上 |
| **柱头** | 小，有粘性，包裹在花内 | 大，羽毛状，悬挂在花外 |
| **花粉数量** | 中等 | 极大 |
| **花粉粒** | 较大，有粘性，带刺或有纹饰以附着在昆虫身上 | 小、轻、光滑、干燥——易被风吹走 |
| **示例** | 玫瑰、香豌豆、毛地黄、毛茛、向日葵 | 禾草、玉米、小麦、橡树、桦树 |

#### 16.3.4 传粉（核心内容）

**传粉** 是花粉粒从 **花药**（雄性部分）传递到 **柱头**（雌性部分）的过程。

**自花传粉（补充内容）**：花粉粒从一朵花的花药传到 **同一朵花的柱头** 或传到 **同一植株上另一朵花的柱头**。

**异花传粉（补充内容）**：花粉粒从一朵花的花药传到 **同一物种的不同植株上的一朵花的柱头**。

**自花传粉与异花传粉的比较（补充内容）**：

| 特征 | 自花传粉 | 异花传粉 |
| :--- | :--- | :--- |
| **遗传变异** | 低——后代与亲本更相似 | 高——后代基因各不相同 |
| **对环境变化的适应** | 差——环境变化时存活的可能性较小 | 好——某些个体可能具有适应新条件的性状 |
| **对传粉者的依赖** | 低——即使在没有昆虫或风的情况下也能发生 | 高——依赖传粉者或风在植株间传递花粉 |
| **拓殖能力** | 对传粉者稀少的隔离种群有利 | 需要与其他同种植物邻近 |
| **有害隐性性状的风险** | 较高——有害等位基因可能积累 | 较低——有害等位基因可被来自另一亲本的显性等位基因掩盖 |

#### 16.3.5 植物的受精（核心与补充内容）

**核心内容**：当 **花粉核**（雄配子）与 **胚珠中的一个核**（雌配子）融合时，发生受精。

**植物受精的过程（补充内容）**：

1. **传粉**——花粉粒落到柱头上。
2. **萌发**——花粉粒吸收柱头上的糖溶液并萌发，长出 **花粉管**。
3. **花粉管生长**——花粉管 **沿花柱向下生长** 朝向子房。管分泌酶以消化穿过花柱组织的路径。花粉管核始终位于尖端，指引生长方向。
4. **进入胚珠**——花粉管进入子房，到达胚珠末端一个微小的开口（珠孔）。
5. **受精**——花粉管的尖端破裂，释放出 **雄核**。该核沿花粉管向下移动，与胚珠内的 **卵核融合**，形成 **受精卵**。
6. **受精后变化**——受精卵在种子内发育成 **胚**。胚珠发育为 **种子**。子房发育为 **果实**（包裹种子并帮助其传播）。花的其他部分（萼片、花瓣、雄蕊、柱头、花柱）枯萎并脱落。

#### 16.3.6 种子的萌发（核心内容）

萌发是种子开始生长并发育成新植株的过程。当条件适宜时，种子内的胚打破休眠并恢复生长。

种子萌发所需的三个环境条件是 **水分、氧气和适宜的温度**。

| 条件 | 为什么需要 |
| :--- | :--- |
| **水分** | 用于 **激活酶**，分解子叶（种叶）中储存的养料（淀粉→葡萄糖）。水分也使种子吸水膨胀，种皮破裂。水将溶解的物质运输到正在生长的胚内。 |
| **氧气** | 用于 **有氧呼吸**，释放能量（ATP）供细胞分裂和生长使用。正在生长的胚呼吸速率极高，需要持续供氧。 |
| **适宜的温度** | 为参与呼吸和生长的酶提供 **最适条件** 的温度。太冷 = 酶工作太慢；太热 = 酶变性。多数种子在 20–25℃ 左右萌发良好。 |

**注意**：光照通常 **不是** 萌发所必需的（种子在地下萌发，那里没有光）。然而，有些种子确实需要光照来打破休眠。大纲不要求了解光照的影响。

### 16.4 人类的有性生殖

#### 16.4.1 男性生殖系统（核心内容）

你必须能够在图表上 **识别** 并 **说明** 这些结构的功能。

| 结构 | 功能 |
| :--- | :--- |
| **睾丸**（单数：testis） | 从青春期开始持续产生 **精子**（雄配子）。也产生激素 **睾酮**。位于体外的阴囊中。 |
| **阴囊** | 容纳睾丸并使其位于 **体外** 的皮肤囊。这使睾丸保持在 **略低于体温** 的温度（约 34–35℃），这对正常的精子生成至关重要。在核心体温（37℃）下，精子生成会受损。 |
| **输精管**（vas deferens） | 肌肉质管道，将精子从每个睾丸向上输送到腹腔并通往尿道。射精时，蠕动的收缩波推动精子前进。 |
| **前列腺** | 产生富含营养的液体，与精子混合形成 **精液**。该液体含有酶和糖，为精子提供营养，帮助其游动和存活。前列腺液也有助于中和尿道中残留尿液的酸性以及女性阴道的酸性。 |
| **尿道** | 贯穿阴茎中心的管道。它将精液（射精时）**或** 尿液（排尿时）排出体外。尿道基部的一环肌肉阻止精液和尿液混合——两者从不被同时释放。 |
| **阴茎** | 男性的外部器官，在性交时将 **精子送入阴道**。它含有勃起组织，充血后变硬勃起，得以插入。 |

**注意**：睾丸 = 精子 + 睾酮。前列腺为精子添加液体 → 形成精液。尿道 = 输送精子或尿液的管道（绝不同时输送两者）。

#### 16.4.2 女性生殖系统（核心内容）

你必须能够在图表上 **识别** 并 **说明** 这些结构的功能。

| 结构 | 功能 |
| :--- | :--- |
| **卵巢**（单数：ovary） | 产生 **卵子（ovum）**——雌配子。女性出生时所有未成熟卵子已存在于卵巢中。青春期后，卵子开始成熟（通常每个月经周期一个）。卵巢也分泌女性性激素 **雌激素** 和 **孕酮**。 |
| **输卵管**（Fallopian tubes） | 连接每个卵巢到子宫的管道。靠近卵巢的开放漏斗状末端捕获排出的卵子。内衬 **纤毛细胞**，纤毛摆动产生水流，将卵子沿输卵管推向子宫。**受精通常发生在输卵管内**。 |
| **子宫**（womb） | 中空的肌肉器官，内壁柔软海绵状（子宫内膜）。是受精卵（受精卵）**植入** 并发育为胚胎继而胎儿的地方。如果未受精，子宫内膜脱落并在月经期间排出。 |
| **宫颈** | 子宫下端的肌肉环，形成从阴道进入子宫的入口。它在怀孕期间将发育中的胎儿保持在原位，并分泌黏液栓以防感染。分娩时，宫颈 **扩张**（变宽）让婴儿通过。 |
| **阴道** | 从宫颈通向体外的肌肉管道。性交时 **接收阴茎和精子**。也充当分娩时的 **产道**，以及经血排出的通路。 |

**注意**：卵巢 = 卵子 + 激素（雌激素和孕酮）。受精 → 输卵管。子宫 = 胎儿发育。阴道 = 接收精子 + 产道。

#### 16.4.3 人类的受精（核心内容）

**受精** 是来自 **雄配子**（精子）和 **雌配子**（卵细胞）的 **核融合**。

- 受精通常发生在 **输卵管** 中（通常在靠近卵巢的上三分之一段）。
- 性交时，数百万精子被送入阴道。它们游过宫颈，进入子宫，再向上进入输卵管。
- 只有一个精子能够成功穿透卵细胞膜。一旦进入，卵子表面发生变化（卵黄膜硬化），不再允许其他精子进入。
- 精子核与卵核融合，形成 **受精卵**。
- 受精卵是 **二倍体**——含有一套完整的染色体（人类为 46 条），一半来自母亲，一半来自父亲。

#### 16.4.4 配子的适应性特征（核心内容）

**精子的适应性特征**：

| 适应性特征 | 功能 |
| :--- | :--- |
| **鞭毛（尾部）** | 一条长长的鞭状尾巴，通过摆动推动精子，使其能够 **游向卵子**，穿过女性生殖道 |
| **大量线粒体** | 位于头部后方的中段。它们通过有氧呼吸释放能量（ATP），为鞭毛的摆动提供动力 |
| **顶体中的酶**（头部区域） | 顶体是精子头部顶端一个帽状结构，含有 **消化酶**。当精子接触卵子时，这些酶被释放出来，**消化出一条穿过卵子周围卵黄膜的通道**，使精子得以穿透并受精 |
| **流线型头部** | 减小精子游动时的阻力 |

**卵细胞的适应性特征**：

| 适应性特征 | 功能 |
| :--- | :--- |
| **细胞质中含有能量储存（养料储备）** | 为受精后发育中的受精卵提供营养和能量，供其在沿输卵管移动的同时不断进行细胞分裂 |
| **胶状外被** | 细胞膜外的一层糖蛋白层（透明带），在 **受精后发生变化**，变得对其他精子不可通透，防止多精入卵（多个精子使同一卵子受精） |

#### 16.4.5 雄配子与雌配子的比较（核心内容）

| 特征 | 精子 | 卵子（Ovum） |
| :--- | :--- | :--- |
| **大小** | 非常小（约 45 µm 长）——人体中最小的细胞之一 | 大（直径约 0.15 mm）——肉眼几乎可见。人体中最大的细胞 |
| **结构** | 高度特化：头部（细胞核+顶体）、中段（线粒体）、鞭毛 | 圆形，相对不特化，被胶状外被包围 |
| **运动能力** | 能自主运动——利用鞭毛主动游动 | 不能自主运动——由输卵管内的纤毛和输卵管壁的收缩被动移动 |
| **产生数量** | 青春期后每天产生 **极多**（每次射精约释放 1–3 亿个） | 每个卵巢中有数千个未成熟卵子，但每个月经周期通常只有一个成熟并排出（排卵）。女性一生中大约只有 400–500 个卵子排出 |

#### 16.4.6 早期发育：从受精卵到植入（核心内容）

在输卵管内受精之后：

1. **受精卵** 沿输卵管向子宫移动的过程中（这段旅程约需 3 天），通过 **有丝分裂** 进行反复的细胞分裂。
2. 它变成一个实心的细胞团，称为 **胚胎**。
3. 胚胎到达子宫，并 **植入** 到子宫壁 **厚而海绵状的内膜**（子宫内膜）中。
4. 植入后，胚胎继续生长和发育。一个名为 **胎盘** 的结构在植入处形成，将发育中的胚胎与母体的血液供应连接起来。

#### 16.4.7 胎儿的发育：关键结构（核心内容）

大纲要求识别并说明以下在胎儿发育中的结构的功能。

| 结构 | 功能 |
| :--- | :--- |
| **胎盘** | 由胚胎外层细胞和母体子宫内膜发育而来的临时器官。是母体血液与胎儿血液之间 **物质交换** 的场所。母体与胎儿的血液供应极为贴近但 **绝不混合**。胎盘将氧气和溶解的营养物质（葡萄糖、氨基酸、维生素、矿物质、抗体）从母体输送给胎儿。它将废物（二氧化碳、尿素）从胎儿血液中排入母体血液。母体随后将这些废物排出体外。胎盘也产生激素 **孕酮** 以维持妊娠。 |
| **脐带** | **物理连接** 胎儿与胎盘的索状结构。它包含脐动脉和脐静脉——将胎儿的血液运入和运出胎盘。脐带内没有神经，所以婴儿出生后剪断脐带不会疼痛。 |
| **羊膜囊**（amnion） | 一层坚韧的膜，包围并包裹发育中的胚胎/胎儿，形成一个密封的、充满液体的囊袋 |
| **羊水** | 羊膜囊内的液体，主要来自母体血浆。它 **缓冲保护发育中的胎儿免受母体腹部碰撞和颠簸**，**允许胎儿自由活动**（促进骨骼和肌肉发育），**维持胎儿周围恒定的温度**，并充当 **抗感染的屏障** |

**需牢记的关键点**：胎盘允许物质的交换——氧气、葡萄糖、氨基酸由母体进入胎儿；二氧化碳和尿素由胎儿进入母体。两套血液供应非常接近但 **不混合**。

#### 16.4.8 胎盘作为交换器官及其局限性（补充内容）

胎盘适应高效的物质交换：

- **巨大的表面积** 用于扩散（胎盘胎儿部分的绒毛膜绒毛极大地增加了表面积）
- **母体与胎儿血液之间的屏障很薄**（仅几个细胞的厚度）

从 **母体 → 胎儿** 穿过的物质：氧气、葡萄糖、氨基酸、维生素、矿物质、抗体（IgG）、水

从 **胎儿 → 母体** 穿过的物质：二氧化碳、尿素（废物）

转运机制：大多数物质通过 **扩散**（沿浓度梯度）穿过；有些（如氨基酸）通过主动运输进行更高效的转移。

**屏障功能**：胎盘通常充当选择性屏障。然而，**某些病原体和毒素可以穿过** 胎盘并影响胎儿：

- **可能穿过的病原体**：风疹（德国麻疹）病毒、HIV、寨卡病毒、弓形虫
- **可能穿过的毒素**：尼古丁（来自香烟烟雾）、酒精、某些药物（如海洛因、可卡因、沙利度胺）
- 这就是为什么孕妇被建议不要吸烟（尼古丁减少对胎儿的供氧，可导致低出生体重）、不要饮酒（酒精可导致胎儿酒精综合征，引起脑损伤和面部畸形），并在怀孕期间避免使用某些药物。

### 16.5 人类的性激素

#### 16.5.1 激素与青春期（核心内容）

**青春期** 是人类发育中 **第二性征** 开始出现的阶段。这在 **性激素** 的控制下发生。

**第一性征** 是出生时就存在的生殖器官（如男性的阴茎和睾丸；女性的卵巢和子宫）。

**第二性征** 是青春期儿童向成人过渡时发生的身体变化。它们受激素释放的控制：女性为 **雌激素**，男性为 **睾酮**。

**睾酮**（男性性激素）：
- **产生部位**：**睾丸**
- **主要作用**：控制青春期男性第二性征的发育
- **效应**：刺激精子生成；导致声音低沉；面部、阴部和体毛生长；肌肉质量和体型增加；阴茎和睾丸生长

**雌激素**（女性性激素）：
- **产生部位**：**卵巢**
- **主要作用**：控制女性第二性征的发育并调节月经周期
- **效应**：刺激乳房发育；启动月经周期；阴毛和腋毛生长；臀部变宽；臀部和股部脂肪分布变化

两性在青春期都会发生一些共同的变化：性器官的生长、体毛的生长以及情绪变化（对性的兴趣增加、情绪波动）。

#### 16.5.2 月经周期（核心内容）

**月经周期** 是一个大约 **28 天**（平均）的月度过程，涉及激素变化、排卵以及子宫为可能的妊娠做准备。它始于青春期（约 12 岁），一直持续到更年期（约 45–55 岁）。

月经周期从 **卵巢的变化** 和 **子宫内膜的变化** 来描述。

| 天数范围 | 卵巢的变化 | 子宫内膜的变化 |
| :--- | :--- | :--- |
| **第 1–5 天**（月经期） | 卵巢处于静息状态；激素水平低 | 子宫内膜 **崩解并脱落**，作为经血通过阴道排出（俗称“例假”） |
| **第 6–13 天**（卵泡期） | 一个 **卵泡**（包裹未成熟卵子的囊）在卵巢中发育。卵泡产生雌激素 | 子宫内膜在雌激素的作用下 **开始再次增厚**——变得海绵状并富含血管，为可能的植入做准备 |
| **第 14 天**（排卵期） | **排卵**：一个成熟卵子从卵巢排出。卵子被输卵管漏斗状的末端扫入输卵管内 | 内膜此时已 **厚且富含血管**，准备好接纳受精卵（受精卵） |
| **第 15–28 天**（黄体期） | 排卵后，空的卵泡转变为 **黄体**。黄体分泌孕酮 | 内膜在孕酮的影响下 **保持厚实** 且血供良好，随时准备植入。如果未受精，黄体破裂（通常在第 26–28 天），孕酮水平下降，内膜不再得到维持 → 这触发了新周期的开始（第 1 天：月经来潮）。 |

**如果发生受精**：受精卵植入厚厚的子宫内膜。发育中的胚胎产生一种激素（hCG——人绒毛膜促性腺激素），使黄体继续存活，因此孕酮水平保持在高位，内膜得以维持。不发生月经——这是怀孕最早的标志。

**如果未受精**：黄体崩解。激素水平下降。子宫内膜脱落 → 月经来潮，周期重新开始。

#### 16.5.3 月经周期的激素调控（补充内容）

月经周期受四种激素的控制。你必须知道它们的 **产生部位** 和 **作用**。

| 激素 | 产生部位 | 主要作用 |
| :--- | :--- | :--- |
| **FSH**（促卵泡激素） | **垂体**（脑部） | (1) 刺激卵巢中 **卵泡的发育**。(2) 刺激发育中的卵泡产生 **雌激素** |
| **LH**（黄体生成素） | **垂体**（脑部） | (1) 大约在第 14 天触发 **排卵**（成熟卵子的排出）。(2) 刺激空卵泡发育为 **黄体**，产生孕酮。LH 由雌激素高峰触发而大量释放 |
| **雌激素** | 卵巢中 **发育中的卵泡**（月经周期中）；**胎盘**（孕期） | (1) 刺激月经后 **子宫内膜的增厚** 和修复。(2) **抑制 FSH 的进一步分泌**（负反馈——阻止同一周期中更多卵泡发育）。(3) 在排卵前处于高水平时，雌激素 **刺激 LH 的释放**（正反馈），触发排卵 |
| **孕酮** | 卵巢中的 **黄体**（月经周期中）；**胎盘**（孕期） | (1) 在周期后半段（第 14–28 天）**维持厚实的子宫内膜**，以便受精卵能够植入。(2) **抑制 FSH 和 LH**（负反馈——孕期防止排卵） |

这四种激素在一个严密调控的序列中相互作用：

1. **FSH** 由垂体释放 → 刺激 **卵泡发育**。
2. 发育中的 **卵泡分泌雌激素** → 子宫内膜增厚。
3. 雌激素升至峰值 → 触发 **LH 大量释放** → **排卵**（第 14 天）。
4. 破裂的卵泡变为 **黄体** 并分泌 **孕酮** → 维持子宫内膜。
5. 如果怀孕，黄体继续产生孕酮，直到胎盘接替这一功能。
6. 如果未孕，黄体崩解 → 孕酮下降 → **月经来潮**。

### 16.6 性传播感染 (STIs)

#### 16.6.1 什么是性传播感染？（核心内容）

**性传播感染 (STI)** 是主要通过 **性接触**（阴道、肛门或口腔性交）传播的感染。病原体在性活动中从已感染的人传给未感染的人。

#### 16.6.2 HIV 与艾滋病（核心内容）

- **HIV**（人类免疫缺陷病毒）是一种 **病原体**（病毒），可引起性传播感染。
- 如果不治疗，HIV 感染可能导致 **艾滋病 (AIDS)**（获得性免疫缺陷综合征）。艾滋病是 HIV 感染的终末期和最严重阶段，此时免疫系统已严重受损，人变得容易罹患健康人免疫系统通常能控制的各种机会性感染和某些癌症。

**HIV 如何作用**：HIV 感染并破坏 **辅助性 T 淋巴细胞**（一种对协调免疫反应至关重要的白细胞）。随着这些细胞数量的下降，免疫系统逐渐削弱，使身体无法抵抗原本无害的感染。

**HIV 的传播方式**：
- 与感染者进行 **无保护的性交**（最常见的方式）
- **共用被感染血液污染的针头或注射器**（例如静脉注射吸毒）
- **由母亲传染给孩子**——怀孕期间穿过胎盘、分娩过程中（接触感染的血液和体液），或母乳喂养期间通过母乳
- **输入受感染的血液**（如今在献血进行筛查的国家已极为罕见）
- **被污染的手术或牙科器械**（在正确实施灭菌操作的地方极为罕见）

**注意**：HIV **不会** 通过日常接触传播，如拥抱、握手、共享食物或饮料、使用同一马桶、蚊虫叮咬或接吻（除非双方都有开放性口疮或牙龈出血）。

#### 16.6.3 控制性传播感染的传播（核心内容）

可以通过以下方式控制性传播感染的传播：

1. **性交时使用避孕套**（屏障法）——避孕套防止与感染的体液（精液、阴道液、血液）直接接触，如果正确并坚持使用则非常有效。
2. **限制性伴侣的数量**——减少伴侣数量可降低遇到感染者的几率。
3. **定期进行 STI 检测和治疗**——早期发现和治疗 STI 可防止进一步传播。
4. **教育和提高意识**——教导人们有关 STI 的风险、传播方式以及如何保护自己。
5. **不共用针头**——为注射吸毒者提供清洁的针具交换项目。
6. **对捐献的血液进行 HIV 和其他血源性病原体的筛查**。
7. **对 HIV 阳性者进行抗逆转录病毒治疗 (ART)**——将病毒载量降低到检测不到的水平，从而大幅降低继续传播的风险。
8. **预防母婴传播**——为 HIV 阳性孕妇提供抗逆转录病毒药物，降低怀孕、分娩或母乳喂养期间将 HIV 传染给婴儿的风险。

| **克隆 (Clone)** | 与产生它的亲本基因完全相同的生物体或细胞。 |
| **有性生殖 (Sexual reproduction)** | 通过两个配子（性细胞）的核融合形成受精卵，产生彼此基因不同的后代的过程。 |
| **配子 (Gamete)** | 性细胞（雄性为精子，雌性为卵子/卵细胞）。配子是单倍体——含有正常染色体数目的一半。 |
| **受精 (Fertilisation)** | 雄配子与雌配子的核融合形成受精卵的过程。 |
| **受精卵 (Zygote)** | 受精过程中由两个单倍体配子融合形成的二倍体细胞。是新生物体的第一个细胞。 |
| **单倍体 (Haploid)** | 含有一套染色体 (n) 的细胞核。配子是单倍体。人类单倍体数目 = 23。 |
| **二倍体 (Diploid)** | 含有两套染色体 (2n) 的细胞核。受精卵和体细胞是二倍体。人类二倍体数目 = 46（23 对）。 |
| **传粉 (Pollination)** | 花粉粒从花药（雄性部分）传递到柱头（雌性部分）的过程。 |
| **自花传粉 (Self‑pollination)** | 花粉粒从一朵花的花药传到同一朵花的柱头，或传到同一植株上另一朵花的柱头。 |
| **异花传粉 (Cross‑pollination)** | 花粉粒从一朵花的花药传到同一物种的不同植株上一朵花的柱头。 |
| **胚（植物）(Embryo)** | 种子内由受精卵发育而来的正在发育的植物。 |
| **胚胎（人类）(Embryo)** | 人类发育的早期阶段，从受精卵第一次分裂至约 8 周；此后称为胎儿。 |
| **植入 (Implantation)** | 胚胎将自己埋入子宫厚而海绵状的内膜的过程。 |
| **胎盘 (Placenta)** | 怀孕期间发育的临时器官。它实现母体血液与胎儿血液之间的物质交换（氧气、营养物质、废物）。两套血液供应不混合。 |
| **脐带 (Umbilical cord)** | 物理连接胎儿与胎盘的索状物；包含将胎儿血液运入和运出胎盘的血管。 |
| **羊膜囊 (Amniotic sac)** | 坚韧、充满液体的膜，包围并保护发育中的胚胎/胎儿。 |
| **羊水 (Amniotic fluid)** | 羊膜囊内的液体，缓冲保护胎儿，允许其活动，维持温度，并防止感染。 |
| **月经 (Menstruation)** | 当受精未发生时，增厚的子宫内膜每月通过阴道脱落的过程（“例假”）。 |
| **排卵 (Ovulation)** | 成熟卵子从卵巢排出的过程（月经周期第 14 天左右）。 |
| **FSH（促卵泡激素）** | 垂体产生的一种激素，刺激卵巢中卵泡发育和雌激素产生。 |
| **LH（黄体生成素）** | 垂体产生的一种激素，触发排卵并刺激黄体产生孕酮。 |
| **雌激素 (Oestrogen)** | 卵巢卵泡产生的女性性激素。刺激子宫内膜增厚并触发 LH 高峰。 |
| **孕酮 (Progesterone)** | 黄体（及孕期胎盘）产生的女性性激素。维持厚厚的子宫内膜。 |
| **睾酮 (Testosterone)** | 主要的男性性激素，由睾丸产生。控制男性第二性征的发育。 |
| **第二性征 (Secondary sexual characteristics)** | 青春期发生的、不直接与生殖相关的身体变化（例如声音低沉、体毛、乳房发育）。 |
| **STI（性传播感染）** | 由病原体引起的感染，通过性接触从已感染者传至未感染者。 |
| **HIV（人类免疫缺陷病毒）** | 一种感染并破坏辅助性 T 淋巴细胞的病毒，导致免疫系统进行性削弱。HIV 是一种可导致艾滋病的性传播感染。 |
| **AIDS（获得性免疫缺陷综合征）** | HIV 感染的晚期阶段，此时免疫系统已严重受损，机体无法抵抗机会性感染。 |

## 第 4 部分：常见考试错误与纠正

| 常见错误 | 正确答案 |
| :--- | :--- |
| 说受精发生在子宫内 | 受精发生在 **输卵管** 内 |
| 混淆胎盘和脐带 | **胎盘** = 物质交换的器官（营养、气体、废物）。**脐带** = 连接缆线。胎盘允许交换；脐带将胎儿血液运进和运出胎盘 |
| 说母体与胎儿的血液混合 | 血液供应 **不混合**——它们被一层薄膜隔开。物质通过该屏障扩散进行交换 |
| 混淆植物的传粉和受精 | **传粉** = 花粉从花药传到柱头。**受精** = 花粉核与胚珠内卵核融合。传粉必须先于受精发生 |
| 写“花变成果实” | **子房** 发育成果实。**胚珠** 发育成种子。花的其他部分枯萎脱落 |
| 混淆 FSH 和 LH 的作用 | **FSH** = 刺激卵泡发育 + 雌激素产生。**LH** = 触发排卵 + 刺激孕酮产生 |
| 说孕酮使子宫内膜增厚 | **雌激素** 使内膜增厚（第 6–13 天）。**孕酮** 在排卵后维持内膜（第 14–28 天） |
| 说“精子游向卵巢” | 精子游向 **输卵管**（不是卵巢）——受精时卵子位于输卵管内 |
| 混淆自花传粉和异花传粉 | **自花传粉**：同一朵花或同一植株。**异花传粉**：同一物种的不同植株 |
| 说种子萌发需要光 | 种子需要 **水分、氧气和适宜的温度**。光照通常 **不是** 萌发所必需的 |
| 混淆睾酮和雌激素的效应 | **睾酮** = 男性激素（睾丸、精子、声音、毛发、肌肉）。**雌激素** = 女性激素（卵巢、乳房、月经、臀部） |
| 说 HIV 就是艾滋病 | **HIV** 是病毒（病因）。**艾滋病** 是综合征（免疫系统严重受损的晚期状态）。并非每个 HIV 感染者都患有艾滋病 |
| 说性传播感染只能靠避孕套预防 | 避孕套非常有效，但限制性伴侣数量、定期检测、教育以及抗逆转录病毒治疗也是重要方法 |
